# 安卓 APK

这个安卓 App 用手机麦克风监听声音，并通过蓝牙向 ESP32 开发板发送和网页一致的舵机律动指令。

## 默认连接对象

App 会扫描带有以下蓝牙服务的开发板：

```text
6e400001-b5a3-f393-e0a9-e50e24dcca9e
```

也就是本仓库 ESP32 固件广播的服务。默认设备名是：

```text
Music Servo ESP32
```

## 功能

- 麦克风监听
- 蓝牙连接 ESP32
- 噪声门限保护
- 灵敏度、输入增强、角度范围、响应速度和平滑程度调节
- 测试舵机按钮

## 构建 APK

如果电脑已经安装 Android Studio，可以直接打开 `android-app` 文件夹，然后构建 APK。

也可以在 GitHub Actions 里自动构建，构建产物会生成 debug APK。

## 使用建议

第一次测试前，先把“噪声门限”调到 40% 左右。安静时确认不会发送指令，再连接舵机并慢慢降低门限。
